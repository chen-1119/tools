# Tools Hub

面向软件工具号的公众号文章和软件链接管理网站，支持文章分类、软件官网、网盘下载、备用链接和审核管理。

## 功能范围

- 公众号文章库：搜索、分类、同步状态和文章详情。
- 软件链接挂载：每篇文章可附带软件官网、夸克、百度、阿里云盘、123 网盘、备用链接等入口。
- 软件链接库：把文章里的软件链接集中展示，方便按链接类型维护。
- 分类运营：AI 工具、办公效率、安卓工具、剪辑设计、开发调试、模板资料。
- 同步录入：录入公众号标题、原文链接、软件名称、分类、软件链接、提取码和审核备注。
- 审核边界：只挂来源清楚、授权明确或允许分发的软件链接，链接可先待配置。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 线上部署

仓库推送到 `chen-1119/tools` 后，GitHub Actions 会构建并发布到 GitHub Pages。

默认访问地址：

```text
https://chen-1119.github.io/tools/
```
