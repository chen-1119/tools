# Tools Hub

面向软件工具号的公众号文章同步网站，支持文章分类、网盘资料挂载和审核管理。

## 功能范围

- 公众号文章库：搜索、分类、同步状态和文章详情。
- 网盘资料挂载：每篇文章可附带夸克、百度、阿里云盘、123 网盘等资料位。
- 分类运营：AI 工具、办公效率、安卓工具、剪辑设计、开发调试、模板资料。
- 同步录入：录入公众号标题、原文链接、分类、网盘链接、提取码和审核备注。
- 审核边界：只挂自有、授权或允许分发的资料，资料链接可先待补。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 线上部署

仓库推送到 `chen-1119/tools` 后，GitHub Actions 会构建并发布到 GitHub Pages。

默认访问地址：

```text
https://chen-1119.github.io/tools/
```
