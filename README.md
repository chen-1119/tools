# Tools Hub

面向软件工具号的自有内容库、公众号文章承接页和选题审核前台。

## 功能范围

- 站内方案首页：搜索、分类、平台筛选、热门内容包卡片。
- 文章承接页：把公众号文章中的工具合集转成站内教程、模板、清单和 FAQ。
- 线索入口：收集用户需求、使用场景和内容类型，不直接上架第三方链接。
- 合规模块：展示版权处理、投稿审核和不托管文件的站点规则。
- 运营线索：记录公众号、短视频、评论区等渠道的用途与风险。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 线上部署

仓库推送到 `chen-1119/tools` 后，GitHub Actions 会构建并发布到 GitHub Pages。

默认访问地址：

```text
https://chen-1119.github.io/tools/
```
