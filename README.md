# Tools Hub

面向软件工具号的合规资源导航、公众号文章承接页和投稿审核前台。

## 功能范围

- 工具资源首页：搜索、分类、平台筛选、热门工具卡片。
- 文章承接页：把公众号文章中的工具合集转成站内专题入口。
- 投稿入口：收集工具名称、官网、授权说明和风险声明。
- 合规模块：展示版权处理、外链审核和不托管文件的站点规则。
- 运营线索：记录论坛、公众号、社群等投放渠道的用途与风险。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 线上部署

仓库推送到 `chen-1119/tools` 后，GitHub Actions 会构建并发布到 GitHub Pages。

默认访问地址：

```text
https://chen-1119.github.io/tools/
```
